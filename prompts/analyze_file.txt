5. **RETRIEVAL-SENSITIVE FIELDS** (CRITICAL):
   Identify fields where semantic search will FAIL because embeddings treat similar values as identical.
   
   Ask yourself for each field:
   - "If a user queries for a SPECIFIC value in this field, will embeddings distinguish it from similar values?"
   - "Do values in this field differ by only 1-2 characters, digits, or small variations?"
   
   Common retrieval-sensitive field types:
   - DATES: "December 3" vs "December 8" are ~99% similar to embeddings
   - CODES/IDs: "IMB45-66-5111" vs "IMB45-66-5117" differ by 1 character
   - ORDER NUMBERS: "254980" vs "254981" are semantically identical
   - SEQUENTIAL NUMBERS: Any incrementing IDs, job numbers, batch numbers
   - MACHINE NAMES: "CLIFFORD MMW" vs "CLIFFORD MWPA" are very similar
   - LOCATIONS: Specific sites that must be distinguished

6. **Canonical Naming**: What variations exist that should map to canonical names?

7. **Suggested Phrases**: Domain-appropriate relationship phrases for the narrative.
